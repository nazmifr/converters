Convertisseur de condensateur JavaScript

Appuyez sur CTRL+S ou cliquez ici (clic droit > enregistrer le lien sous) pour télécharger une version fonctionelle hors-ligne de cette page.

Cette page ne vous piste pas, ne contient pas de cookies elle nécessite juste le code javascript contenu dans la page HTML.

Le code de la page est open-source par design et aucune restriction en dehors de l'attribution existe dessus. Vous pouvez en faire ce que vous voulez même un usage commercial tant que l'attribution en bas reste.

Merci aux développeurs talentueux de faire des forks mieux conçus :p

Ce convertisseur peut donner le code à trois chiffres des condensateurs de 1pF à 990uF (généralement on rencontre des condensateurs céramique de 1pF à 10uF, rarement plus ou moins)

Entrer une valeur suivie de pF nF uF mF (l'unité par défaut est le nF)

pF nF uF mF F

012
